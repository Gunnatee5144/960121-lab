If the search box is empty and no category is selected

Show the full product list. That is the default state, so the page should render all products instead of filtering them away.

If the search returns no results

Show a clear empty-state message such as No matching products found so the user understands the filter worked but nothing matched.

Code Snippet Description & Explanation
HTML STRUCTURES
<!DOCTYPE html>
<html lang="en">
  <head>...</head>
  <body>...</body>
</html>
Standard Document Skeleton: The fundamental structure of every HTML page. The DOCTYPE tells the browser we are using HTML5, while html, head, and body define the root, metadata, and visible content respectively.
<div class="container">
  <h1>Title</h1>
  <p>Content...</p>
</div>
Semantic Containers: Using <div> elements to group related content. This allows developers to apply specific styling or behavior to a block of elements collectively.
CSS STYLING
.card {
  display: flex;
  justify-content: center;
  align-items: center;
}
Flexbox Layout: A powerful 1D layout model. Using display: flex allows for easy alignment and distribution of space among items within a container, even when their size is unknown.
@media (max-width: 768px) {
  .nav { display: none; }
}
Media Queries: The heart of responsive design. These allow you to apply different styles based on the device's screen size or resolution, ensuring your site looks great on phones, tablets, and desktops.
BOOTSTRAP FRAMEWORK
<div class="row">
  <div class="col-md-6">Left</div>
  <div class="col-md-6">Right</div>
</div>
Grid System: Bootstrap's mobile-first flexbox grid. It uses a 12-column system. col-md-6 means the element will take up 6 columns (50% width) on medium-sized screens and larger.
<button class="btn btn-primary">
  Submit
</button>
Component Utility: Bootstrap provides pre-styled components. btn establishes the base button style, and btn-primary applies the primary theme color (usually blue) and hover effects.
